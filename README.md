# homebrew-opendisplay

Homebrew tap for [OpenDisplay](https://github.com/sahilmishra0012/OpenDisplay) — an open-source macOS display management utility.

## Install

```bash
brew tap sahilmishra0012/opendisplay
brew install --cask opendisplay
```

## Update

```bash
brew upgrade opendisplay
```

## Uninstall

```bash
brew uninstall opendisplay
```
